# VULNER - Network Scanner & Vulnerability Mapper

## Overview
Automated network reconnaissance and vulnerability mapping tool. Developed during the Network Security course (ZX301) to streamline the initial phases of penetration testing.

## Key Features
* **Automated Reconnaissance:** Executes Nmap scans with multi-level configurations (Basic/Full).
* **Credential Testing:** Integrates Hydra to test weak credentials on common services (FTP, SSH, Telnet, RDP).
* **Vulnerability Analysis:** Automatically maps open ports to known CVEs using Searchsploit.
* **Structured Reporting:** Packages all scan logs and vulnerability reports into a unified ZIP archive.

## Ecosystem Impact
This tool provides a practical, educational workflow for students to understand how different security tools can be chained together. It bridges the gap between theoretical knowledge and practical network security operations.

## Disclaimer
For educational purposes only. Use only in authorized lab environments.
